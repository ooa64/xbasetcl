# xbasetcl

Tcl interface to the XBASE library (http://linux.techass.com/projects/xdb).

xbasetcl 1.0 alpha 3

Tested: 

    Xbase: 2.1 cvs snapshot from 2002/09/09.
    Os/compiler: Windows/msvc, Linux/gcc.
    Tcl: 8.0.5, 8.3.4.
